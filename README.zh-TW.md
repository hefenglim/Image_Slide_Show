# Image SlideShow 照片輪播 — 具備回憶探索功能的智慧桌面相簿

> 一款輕量的 Windows 桌面照片輪播工具。與一般輪播不同的是，Image SlideShow 讓你在**隨機與順序模式之間一鍵無縫切換** — 隨機播放時若被某張照片觸動回憶，可以立刻切到順序模式翻閱同一場活動、同一趟旅程前後的照片，看完後再切回隨機繼續探索。拖放即用、無邊框設計，適合用作 **電子相框**、**個人回憶探索器** 或 **乾淨的展示工具**。

**關鍵字：** `照片輪播` `照片播放器` `隨機播放` `桌面幻燈片` `電子相框` `照片輪播軟體` `拖放輪播` `無邊框輪播` `全螢幕照片瀏覽` `回憶探索`

## 功能特色

### 🖼️ 核心功能
- **拖放匯入：** 將資料夾或照片檔案直接拖曳進視窗即可開始輪播。
- **支援格式：** `JPG`, `JPEG`, `PNG`, `GIF`, `BMP`, `TIF`。
- **INI 播放清單：** 可直接載入預先建立的 `.ini` 照片清單檔案。

### 🎮 快速操控
| 動作 | 快捷鍵 / 滑鼠 |
|---|---|
| **播放 / 暫停** | `S` |
| **切換隨機 / 順序播放** | `空白鍵` |
| **下一張 / 上一張** | `→` `←` 方向鍵 或 `滑鼠滾輪` |
| **切換乾淨展示模式** | `點擊圖片` |
| **進入 / 退出全螢幕** | `F` 或 `F11` |
| **退出展示 / 關閉程式** | `Esc` |

### 🧠 智慧播放

#### 🔀 隨機模式 (Shuffle)
- 照片會被**隨機選取且不重複**，直到所有照片都播放過一輪。
- 全部播放完畢後會自動重置（螢幕會顯示「Random Cycle Reset」提示）。
- 使用 `←` / `→` 可在**最近 32 張隨機播放過的照片**之間來回瀏覽。

#### ▶️ 順序模式 (Sequential)
- 照片依檔案/資料夾順序依序播放。
- 使用 `←` / `→` 可自由地在照片清單中往前或往後瀏覽。

#### 🔄 無縫模式切換 — 沉浸式回憶探索
你可以隨時按 `空白鍵` 在隨機與順序模式之間切換：

- **隨機 → 順序：** 畫面停留在當前照片，此時 `←` / `→` 會讓你瀏覽該照片在原始資料夾中的**前後鄰居** — 非常適合翻閱同一場活動或旅程中的其他照片。
- **順序 → 隨機：** 程式會回到你切換前在隨機歷史紀錄中的位置，從那裡繼續隨機播放。

> **使用情境：** 隨機播放到一張令人懷念的旅遊照片 → 按 `空白鍵` 切到順序模式 → 用方向鍵翻閱那天拍的所有照片 → 再按 `空白鍵` 切回隨機播放，從剛才的位置繼續。

#### 其他播放功能
- **自訂間隔：** 設定每張照片的顯示秒數。
- **螢幕提示 (OSD)：** 播放、暫停、切換模式等操作時會顯示即時文字提示。

### 📋 選項選單（透過狀態列存取）
| 選項 | 說明 |
|---|---|
| **Set Duration** | 設定輪播間隔秒數 |
| **Clear All Images** | 清除所有已載入的照片 |
| **Save Image List** | 將目前播放清單匯出為 `.ini` 檔案，以便日後載入 |
| **Show Taskbar** | 切換是否顯示在 Windows 工作列 |
| **Shuffle** | 切換隨機 / 順序播放 |
| **Fullscreen** | 切換全螢幕 |
| **Always Top** | 讓視窗保持在最上層 |
| **About** | 顯示版本資訊、聯絡信箱與贊助連結 |

### ⚙️ 系統整合
- **記住你的偏好：** 自動將設定（間隔秒數、隨機模式、全螢幕、最上層顯示、工作列顯示）儲存至 `config.ini`。
- **最上層顯示：** 讓輪播視窗保持浮在其他視窗上方。
- **無邊框視窗：** 展示模式下視窗沒有標題列，但你仍然可以拖曳視窗以及從任何邊緣調整大小。

## 技術概覽

- **語言：** C#
- **框架：** .NET Windows Forms
- **UI 與互動：** 自定義無邊框視窗拖曳與縮放，直接在滑鼠事件中實作。
- **狀態管理：** 使用自訂 `IniRW` 類別讀寫應用程式狀態。
- **贊助模組：** 內建 PayPal 贊助功能 (`PayPalDonate`)。

## 開始使用

1. 使用你偏好的 IDE（例如 Visual Studio）開啟 `Image_SlideShow.sln`。
2. 建置解決方案。
3. 執行應用程式。
4. 將任何照片檔案或包含照片的資料夾直接拖放到視窗中，即可開始輪播！

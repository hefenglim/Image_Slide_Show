A simple & basic photo slideshow widget application, for use in many sets of photo run slideshow in one screen.

----------------------------------------------------------------

Feature:
# Just drag all photos and start slideshow.

# Open multiple windows for more than one sets slideshow.

# Click photo for frameless slideshow display.

----------------------------------------------------------------

Use Guide:
# Drag photos or saved lists file into frame to start slideshow.

# Click photo for Frameless slideshow.

# Press any key to pause slideshow, more than 2 windows slideshow will sync pause.

# [Option] for save lists & misc setting.
